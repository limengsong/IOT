#### 检查打印机状态

```
url：http://lms-pc:1880/check
type：POST
body：
["ZDesigner ZT210-300dpi ZPL","OneNote","HWSealPrinter"]
(说明：body传递打印机名字，可一个可多个，数组形式)

返回值：
[
    {
        "name": "ZDesigner ZT210-300dpi ZPL",
        "status": "available"
    },
    {
        "name": "OneNote",
        "status": "unavailable"
    },
    {
        "name": "HWSealPrinter",
        "status": "unavailable"
    }
]
```
